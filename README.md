# UDM CETVent
 
An event website for a College of Engineering is an essential resource designed to connect students, faculty, alumni, and industry professionals with a wide range of activities and opportunities within the engineering community. This platform typically features an organized event calendar that lists upcoming workshops, guest lectures, career fairs, and conferences, making it easy for users to find relevant information.

A drafted project; where students of College of Engineering in Universidad de Manila can freely express everything in their own space and a computing where they can be understood.
